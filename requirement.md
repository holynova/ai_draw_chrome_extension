# 功能
- 创建一个chrome插件, 为用户方便的创建AI绘画提示词
- 提示词由两个元素组成:画面主体和绘画风格, 两者组合形成一个绘画提示词, 供用户复制或者直接输入到gemini/ChatGPT网页的输入框中
- 绘画风格:
    - 从prompts.json中取. 
    - 绘画风格按分组形成选项, 用户可以手动点击选取
- 画面主体: 
    - 利用网络, 总结一个网络常用的画面元素分组, 比如人物,动物,环境,水果,物体,建筑等分组, 每个分组生成至少10个可选的元素, 生成一个json文件在源码中, 供插件引用. 
    - 画面主体按分组形成选项, 用户可以手动点击选取.
    - 有一个输入框, 用户可以手动输入自己的画面主体
- 随机按钮: 启动动画, 随机从画面主体和绘画风格中选一个, 随机功能要有酷炫的抽奖动画, 抽出两个元素
- 点击复制按钮, 将绘画提示词复制到剪贴板
- 点击输入按钮, 如果当前页面是gemini或者gpt,将绘画提示词输入到gemini/ChatGPT网页的输入框中


# 技术规范
- 用WXT创建chrome插件
- 用react+ts+windsurf+tailwind+pnpm
